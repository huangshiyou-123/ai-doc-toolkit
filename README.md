# AI 文档处理工具箱

> 一套用 AI 辅助文档处理的 Prompt 模板工具包——把杂乱的信息变成结构化的输出。

## 这是什么

项目助理、产品助理、行政岗位的日常工作中，大量时间花在"信息整理"上——客户说的需求要整理成文档、长篇文章要提炼要点、会议记录要归纳成纪要。这些工作本质上都是"吃进去模糊信息，吐出来结构化文档"。

这个工具箱把我在实践中验证过的 Prompt 模板整理成模板，配上使用说明和效果演示，让 AI 能够按照固定格式输出，提高文档处理的质量和效率。

## 包含什么

| 文件 | 用途 |
|------|------|
| `需求整理助手.txt` | Prompt 模板：把客户杂乱需求 → 结构化需求文档 |
| `文档摘要助手.txt` | Prompt 模板：把长文章 → 要点摘要 + 关键数据 |
| `演示材料.pdf` | 实际使用效果演示——输入一段客户原话，看 AI 输出什么 |
| `README.md` | 项目说明文档（你在读的这个） |

## 怎么用

1. 打开你常用的 AI 工具（ChatGPT / Claude / Kimi / DeepSeek）
2. 复制对应 `.txt` 文件中的 Prompt 模板
3. 在分割线下方粘贴你要处理的内容
4. AI 按设定格式输出结构化结果

## 演示效果

**输入（一段客户原话）**：

> "我们公司想做一个员工考勤小程序，跟钉钉打卡数据打通，员工能在小程序上看到自己每天的打卡记录和加班时长。月底自动生成考勤汇总表，HR可以直接导出Excel。审批流程也要走起来，请假、加班申请都能在小程序里批。"

**AI 输出（自动拆解后的结构化文档）**：

- 识别出 **1 条核心需求**
- 拆解出 **5 个功能点**
- 主动提出 **5 个需要跟客户确认的问题**
- 给出 **2 条下一步推进建议**

详细输出见 `演示材料.pdf`。

## 为什么建这个项目

我是计算机网络专业的学生，对 AI 工具在日常办公场景的应用有实际兴趣。课余自己折腾过 AI 开发环境的搭建（DeepSeek + Claude Code），也积累了 Prompt 调试和文档整理的经验。这个项目就是把我日常用的 AI 辅助方法整理成可复用的工具箱——既是给自己梳理思路，也是以后工作中可以直接拿来用的生产力工具。
